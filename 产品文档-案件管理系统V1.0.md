## 一、产品概述

### 1.1 产品定位

本系统是一款专为**法院诉前调解机构**量身定制的全流程数字化管理平台，覆盖案件从接收、分派、处理到结案的全生命周期管理，融合智能数据分析、精细化权限控制、财务一体化管理等先进理念，助力调解机构实现数字化转型，提升案件处理效率与服务质量。

### 1.2 设计理念

- **全流程数字化**: 案件流转无纸化，数据实时同步
- **智能化决策**: 基于大数据的调解员画像与案件分析
- **精细化管控**: 多维度权限体系，确保数据安全与业务隔离
- **便捷化操作**: 简洁直观的用户界面，降低学习成本

### 1.3 业务价值

| 价值维度 | 具体收益 |
|---------|---------|
| 效率提升 | 案件处理周期缩短30%，批量导入/导出支持 |
| 成本控制 | 无纸化办公，减少人工统计与核对工作量 |
| 质量保障 | 调解员六维能力评估，案件质量可追溯 |
| 决策支持 | 实时数据大盘，辅助管理层科学决策 |
| 风险管控 | 精细化权限控制，敏感数据加密存储 |

- 多字段组合模糊查询
- 按时间范围、状态、金额区间筛选
- Excel批量导入/导出

**4. 案件详情全景展示**
- 案件基本信息、时间线、处理记录

### 2.2 调解员画像系统（核心亮点）

#### 功能概述
基于大数据分析的调解员综合能力评估系统，构建六维能力模型，为人员管理、绩效考核、培训提升提供科学依据。

#### 六维能力评估模型

| 维度 | 权重 | 评估逻辑 |
|------|------|---------|
| 调解成功率 | 15% | 成功结案数/总处理案件数 |
| 调解效率 | 15% | 平均调解天数（≤30天满分，分段递减） |
| 复杂度适应力 | 15% | 大额/超大额案件处理能力评估 |
| 案件覆盖度 | 20% | 案由类型多样性（类别数量+分布均匀性） |
| 创收能力 | 20% | 平均调解费水平分级评估 |
| 抗失败能力 | 15% | 可控失败原因占比分析 |

#### 智能评分算法

**案件覆盖度得分算法**（行业首创）:
```
类别数量得分 = min(案件类型数 × 10, 100) × 40%
分布均匀性得分 = 基于相邻案由数量差异计算 × 60%
总得分 = 类别数量得分 + 分布均匀性得分
```

**调解费得分算法**:
```
得分 = 50 + 25 × log10(平均调解费/10000)
```

- 案由分类TOP10柱状图
- 失败原因分析柱状图
- 调解员综合排名榜单

#### 功能概述

#### 管理员大盘

**核心指标**:

**核心指标**:
- 个人案件状态分布
- 即将超时案件预警
- 近30天处理效率趋势

### 2.4 财务一体化管理

#### 功能概述
实现调解费用全流程线上管理，包括银行流水对接、费用结算、发票管理等。

#### 功能模块

**1. 银行流水管理**
- 银行流水Excel批量导入
- 智能匹配关联案件
- 结算/退费状态跟踪
- 流水明细查询与导出

**2. 案件付款流水**
- 多付款方式记录（银行转账/现金/支付宝/微信）
- 付款时间自动记录
- 付款凭证附件管理

**3. 开票管理**
- 发票申请与审核流程
- 开票状态实时跟踪（暂未申请/待开票/已开票）
- 发票PDF在线预览与下载
- 原告/被告费用分摊自动计算

**4. 结案财务归档**
- 结案方式记录（司法确认/撤诉/民初/其他）
- 调解费总额与分摊明细
- 开票金额与未开票金额统计

### 2.7 用户权限管理体系

#### 功能概述
构建多维度、精细化的权限控制体系，确保数据安全与业务隔离。

#### 角色体系

| 角色 | 权限范围 |
|------|---------|
| 总部管理员 | 系统全局管理，所有数据可见 |
| 驻点管理员 | 仅管理所属驻点数据 |
| 调解员 | 个人案件处理，数据大盘查看 |
| 案件助理 | 协助案件处理，受限操作权限 |
| 财务专员 | 财务管理相关功能 |

#### 权限控制维度
- **角色权限**: 功能菜单级控制
- **驻点权限**: 数据隔离（九堡/彭埠/本部/笕桥/丁兰/四季青/凯旋街道/闸弄口）
- **案件来源权限**: 上城法院本部/九堡法庭/笕桥法庭/综治中心

#### 智能映射规则
支持9个归属地映射到4个案件来源，特殊逻辑自动处理（如九堡/彭埠/笕桥且案号含"杭"的统一归为"综治中心"）。

## 三、产品特点与优势

### 3.1 创新性特点

#### 1. 行业首创的调解员六维画像
- 基于大数据的多维度能力评估
- 科学的权重分配与评分算法
- 可视化雷达图直观展示
- 为绩效考核与培训提供数据支撑

#### 2. 智能案件覆盖度评分
- 首创"类别数量+分布均匀性"双维度评估
- 鼓励调解员拓展案由类型
- 算法公平，避免单一类型案件堆积

#### 3. 精细化权限控制体系
- 角色+驻点+案件来源三重权限控制
- 满足不同组织架构的管理需求
- 数据隔离严格，安全保障完善

### 3.2 技术优势

| 优势点 | 说明 |
|--------|------|
| 技术架构先进 | Spring Boot + MyBatis-Plus，业界主流技术栈 |
| 数据安全可靠 | SHA256加密，OSS加密存储，多层安全防护 |
| 系统性能优异 | 支持大数据量案件处理，响应速度快 |
| 扩展性强 | 模块化设计，便于功能扩展与定制开发 |
| 用户体验佳 | 现代化UI设计，响应式布局，操作便捷 |

### 3.3 业务优势

#### 1. 全流程数字化
- 案件流转无纸化，环保高效
- 状态实时同步，避免信息滞后
- 电子归档，便于历史查询

#### 2. 智能化辅助
- 智能分派，优化资源配置
- 即将超时预警，避免案件积压
- 数据分析辅助决策

#### 3. 财务一体化
- 费用计算自动化
- 开票流程线上化
- 财务数据可追溯

### 3.4 竞争优势对比

| 对比维度 | 本产品 | 传统系统 |
|---------|--------|---------|
| 调解员评估 | 六维画像+智能评分 | 简单统计 |
